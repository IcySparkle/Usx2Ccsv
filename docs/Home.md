# UsxToCsv Wiki

Welcome to the UsxToCsv Wiki. This project converts USX/USFM/SFM scripture files into a consistent CSV format suitable for publishing, analysis, and automation.

## What this repo includes
- PowerShell script (Windows-friendly)
- Go CLI (cross-platform binary)
- Rust CLI (cross-platform binary)
- Web server for upload -> convert -> download
- React UI (optional) for a modern frontend
- Docker image build and GHCR publishing

## Start here
- If you want a desktop CLI: see `CLI-Usage.md`
- If you want a web app: see `Web-App-Guide.md`
- If you want hosting: see `Deployment.md`
- If you want automation: see `Automation-Integration.md`
